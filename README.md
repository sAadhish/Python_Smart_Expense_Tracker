💸 Smart Expense Tracker (CLI App)

A clean, modular, and efficient command-line expense tracker built using Python and Object-Oriented Programming (OOP).

⸻

🚀 Overview

Managing daily expenses shouldn’t be messy or complicated.

This project is a lightweight CLI-based Expense Tracker designed to help users:
	•	Track expenses easily 📊
	•	Understand spending patterns 📈
	•	Organize data by category 🧾
	•	Store data persistently using JSON 💾

⸻

🎯 Why This Project?

This project was built to:
	•	Strengthen core Python and OOP concepts
	•	Understand real-world application structure
	•	Practice clean architecture & separation of concerns
	•	Simulate how production-level applications manage data

⸻

✨ Features

✅ Add new expenses (amount, category, date)
✅ View all recorded expenses
✅ Calculate total spending
✅ Category-wise expense summary
✅ Persistent storage using JSON
✅ Clean modular design (OOP-based)

⸻

🧠 Tech Stack
	•	🐍 Python (Core + OOP)
	•	📁 JSON (Data Storage)
	•	🖥️ CLI (Command Line Interface)

⸻

📂 Project Structure

expense-tracker/
│── main.py              # CLI interface (user interaction)
│── expense.py           # Expense class (data model)
│── expenseManager.py    # Business logic
│── storage.py           # Data persistence (save/load)
│── expense_list.json    # Stored expense data

⚙️ How It Works
	1.	User interacts through a CLI menu
	2.	Inputs are converted into Expense objects
	3.	ExpenseManager handles all operations
	4.	Data is stored and retrieved using JSON

⸻
